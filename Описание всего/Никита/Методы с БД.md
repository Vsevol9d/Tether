### Чтобы начать работу с БД, нужно написать:
```python
from Database.api import Session, DataBase

with Session() as session:  # создание сессии
    # создание экземпляра класса, 
    # через который осуществляется вся взаимосвязь с БД
    db = DataBase(session)  
```


## isSuccess: все api-функции БД возвращают isSuccess-словарь с одинаковой структурой
### Если при выполнении функции не возникло ошибки:  
```json lines
{
  'isSuccess': True, 
  'data': <...>
}
``` 
`'data'` может принимать типы `list`, `dict` и `bool` в разных функциях  
Если написано *isSuccess без data*, это значит, что при успехе операции вернётся `{'isSuccess': True}`
### Если при выполнении функции возникла ошибка:
```json lines
{  
  'isSuccess': False,  
  'error': 'краткое сообщение об ошибке, например',
  'long_error': 'сообщение об ошибке с полным Traceback'
}
```

## Есть 5 [базовых методов](../../Database/methods/basic_methods.py), которые работают со всеми таблицами:
Чтобы их вызвать, нужно написать *db.<таблица>.<метод>* (например, db.users.select_all())
  * `add(...)`
  * `update(...)`
  * `delete(...)`
  * `select_all()`
  * `exists(...)`


### Функция add(...) -> dict
*Добавление новой строки с данными в соответствующую таблицу* 

За **параметры** функции брать названия колонок в таблице  
За **аргументы** - значения, нуждающийся во вставке в эту колонку  
*Исключениями* являются колонки `"id"`, `"date_created"`, `"creation_date_time"`, `"joined_at"`  

**Возвращает** словарь isSuccess, где data содержит словарь со всеми колонками в таблице (что-то заполняется автоматически)

<details>
<summary>Пример вызова и результата</summary>

```python
# Вызов
db.users.add(name='Новый аккаунт', username='NewUser', password='New pass', lastname='Новоаккаунтов')

# Возвращаемый словарь
{
    'isSuccess': True, 
    'data': {
        'id': 5, 
        'name': 'Новый аккаунт', 
        'username': 'NewUser', 
        'lastname': 'Новоаккаунтов', 
        'birthday': None,

        'last_time_online': '22-05-2026 21:39:57', 
        'password': 'New pass', 
        'phone': None,

    }
} 
```

### Функция update(self, pk: int | list, **kwargs) -> dict
*Обновление строчки в таблице, через её id*

**pk** - первичный ключ строки Если ПК у строки 2 и более, то брать их все через список
За **параметры** функции брать названия колонок в таблице, которые нужно обновить
За **аргументы** - значения, на которые будут обновленны ячейки  
**Например:** `db.users.update(pk=1, username='updated_user', name='updated_user')`  
обновляет у юзера с `id == 1` обновляет `updated_user` на `'updated_user'` и `name` на `'updated_user'`

**Возвращает** isSuccess, где data содержит словарь со всеми колонками в таблице

<details>
<summary>Пример вызова и результата</summary>

```python

db.users.update(pk=5, lastname='Нормальнофамильев')

# Возвращаемый словарь

    'isSuccess': True, 
    'data': {
        'id': 5, 
        'name': 'Новый аккаунт', 
        'username': 'NewUser',
```

### Функция exists(...) -> dict
**Проверка существования значения в определённой строке таблицы**

За **параметры** функции брать названия колонок в таблице  
За **аргументы** - значения, нуждающийся в проверке наличия в данной колонке  
**Например:** *exists(username='Nikita', name='Nik')*

**Возвращает** isSuccess, гда data содержит:
   * *True*, если значение существует
   * *False*, если значение не существует, или неверное имя колонки

<details>
<summary>Пример вызова и результата</summary>

```python
# Вызов
db.users.exists(username='NewUser', password='New pass')

# Возвращаемый словарь
{
    'isSuccess': True, 
    'data': True
} 
```
</details>

**Возвращает** isSuccess, где в data сохранён словарь с инфой о пользователе в личной переписке
