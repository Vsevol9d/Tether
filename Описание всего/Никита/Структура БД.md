# Database/models/ - Классы с шаблонами таблиц

### класс Users (таблица users):
Инфо обо всех пользователях
  * id
  * name
  * username
  * lastname
  * aavatar_url
  * date_created
  * last_time_online
  * phone
  * email
   
### класс Chats (таблица chats):
Инфо обо всех чатов (схоже по смыслу с таблицей users)
  * id
  * name
  * avatar_url
  * date_created
  
### класс Messages (таблица messages):
Инфо обо всех сообщениях
  * id
  * type (может принимать "text" или "file")
  * text
  * file_id
  * creation_date_time
  * chat_id
  * user_id
    
### класс Participants (таблица participants):
Инфо обо всех пользователях как "участниках чатов"
  * chat_id
  * user_id
  * joined_at
  * role

### класс Friend (таблица friends):
Инфо обо всех связях между пользователями
  *  user_id_1
  * user_id_2
  * level_relationships - может принимать значение 1, 2, 3 или 4, где:
    - 1 - знакомые - 2 пользователя состоят в одной группе
    - 2 - знакомые+ - имеется личная переписка между двумя пользователями
    - 3 - друзья - знают телефон друг друга
    - 4 - лучшие друзья - пользователь сам их помечает (на будущие)
