# Описания методов получения данных из БД
## Переключение на окно с чатами (метод switch_to_chatting_page в классе App, выполняется после регистрации/входа)
При первом переключении получаем данные чатов пользователя для последующего вывода на экран.
### Запросы на получения данных:
* Получение всех чатов пользователя (приминает id пользователя) (возвращает словари с id чата, типом чата, именем чата)
* Получение всех пользователей для каждого чата (принимает id чата) (возвращает словари с id пользователя, именем пользователя для каждого пользователя)
* Получение последнего сообщения и имени его отправителя (принимает id чата) (возвращает строки с последним сообщением и именем его отправителя, если в чате нет сообщений то возвращает 'Начните общение!')
### Дополнительные операции:
* Составление списка участников и списка имён участников во время получения всех пользователей чата
* Составление списка возможных для добавления в чаты пользователей
* Составление словаря доступных чатов где по id чата можно получить: имя: str, отправитель + последнее сообщение: str, количество участников: int, список участников: list[dict с юзерами], список имён участников: list[str], новый ли чат: bool
## Открытие чата (метод open_chat в классе App, выполняется после нажатия на selectable_chat_view)
Принимает id чата, который нужно открыть
### Запросы на получение данных:
* Получение данных сообщений в чате (принимает id чата) (возвращает user_id, file_id, type, text сообщения в виде словаря)
* Получение имени отправителя для каждого сообщения (принимает user_id сообщения) (возвращает единственную строку с именем пользователя)
### Дополнительные операции:
* Запись id чата, типа чата, имени чата, списка имён участников, количества участников в переменную current_chat_info
* Получение типа контента в сообщении проверкой file_id (если None, то text, иначе image)
* Проверка имени отправителя (если id отправителя == id пользователя - сделать 'Вы', иначе оставить как есть)
